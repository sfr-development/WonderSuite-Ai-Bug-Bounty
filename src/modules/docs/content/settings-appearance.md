# Settings — Appearance

The **Appearance** tab customizes how WonderSuite looks. All settings apply instantly and persist across sessions.

## Color Theme

Three built-in themes:

- **Dark** — the default; dark grey surfaces, easy on the eyes for long sessions.
- **Light** — a bright, high-contrast palette.
- **Hacker** — pure black with green-on-black terminal styling.

## Accent Color

The primary highlight color used for active states, selected items, links, and focus rings. Pick from seven presets (amber, blue, purple, green, red, orange, cyan). The change ripples through the whole UI immediately.

## UI Zoom Scale

A slider from **80 %** to **130 %** that scales the entire interface. Bump it up on a high-resolution display, or down to fit more on screen.

## Compact Mode

Reduces margins and padding throughout the app so more data fits in the same space — useful on smaller screens or when working with dense tables like [Traffic](page:traffic) and the [Intruder](page:intruder) results.
