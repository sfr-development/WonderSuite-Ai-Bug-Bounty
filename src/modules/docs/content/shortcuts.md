# Keyboard Shortcuts

WonderSuite's global shortcuts jump straight to a module from anywhere in the app.

## Module navigation

| Shortcut | Module |
|---|---|
| <kbd>Ctrl+1</kbd> | [Dashboard](page:dashboard) |
| <kbd>Ctrl+2</kbd> | [Intercept](page:intercept) |
| <kbd>Ctrl+3</kbd> | [Traffic](page:traffic) |
| <kbd>Ctrl+4</kbd> | [Repeater](page:repeater) |
| <kbd>Ctrl+5</kbd> | [Intruder](page:intruder) |
| <kbd>Ctrl+6</kbd> | [Scanner](page:scanner) |
| <kbd>Ctrl+7</kbd> | [Sitemap](page:sitemap) |
| <kbd>Ctrl+8</kbd> | [Sequencer](page:sequencer) |
| <kbd>Ctrl+9</kbd> | [Tools](page:tools) |
| <kbd>Ctrl+0</kbd> | [Findings](page:findings) |

## Other

| Shortcut | Action |
|---|---|
| <kbd>Ctrl+,</kbd> | [Settings](page:settings-general) |
| <kbd>F1</kbd> | Documentation (this module) |

The sidebar also shows each module's shortcut next to its name when expanded, and in the tooltip when collapsed. Modules without a number shortcut are reached by clicking them in the sidebar.

## In-module shortcuts

Some modules have their own keys:

- **[Repeater](page:repeater)** — <kbd>Enter</kbd> in the URL field sends the request.
- **[WebSocket](page:websocket)** — <kbd>Enter</kbd> in the composer sends a frame; <kbd>Shift+Enter</kbd> inserts a newline.
- Most search/filter boxes act on <kbd>Enter</kbd>.
